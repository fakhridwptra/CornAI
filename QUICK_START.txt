## 🚀 Cara Buka & Run

```
1. Buka Android Studio
2. File → Open → Pilih folder "CornAI"
3. Tunggu Gradle sync selesai
4. Klik Run (▶️) atau Shift+F10
5. Selesai! App jalan!
```
